# 校验相关方法

### isNull

判断是否为 null

**类型声明**

```ts
function isNull(value: any): boolean;
```

**参数**

| 必填 | 参数名 | 类型 | 描述 | 默认值 |
| :-: | --- | --- | --- | --- |
| * | value | any | 待判断的值 | - |

**返回值**: `boolean`

**示例**

```ts
import { isNull } from '@cmtlyt/base';
// import { isNull } from '@cmtlyt/base/utils/verify';

isNull(null); // true
isNull(undefined); // true
isNull(0); // false
isNull(''); // false
isNull([]); // false
isNull({}); // false
isNull(() => {}); // false
isNull(Promise.resolve()); // false
```
